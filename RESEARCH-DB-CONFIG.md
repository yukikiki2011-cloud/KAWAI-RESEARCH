## カラム構成

| カラム | 内容 |
|--------|------|
| メディア | X または note または その他 |
| 投稿日（推定） | YYYY-MM-DD形式 |
| タイトル／本文（抜粋） | 投稿タイトルまたは本文冒頭100文字 |
| カテゴリ | デザイン思考/AI/ビジネス/ブランディング/教育/その他 |
| URL | 投稿のURL |
| メモ・要点 | 要点を1〜2文で |
| チェックすべき関連サイト | 投稿内で言及されているサイト・ツール |
| 重要なワード | キーワード（カンマ区切り） |
| 今後深堀すべきポイント | フォローすべき視点・問い |

---

## 自動更新スケジュール

毎日 **23:00 JST** にリモートエージェントが、各メンバーの新着投稿をリサーチして追記します。
